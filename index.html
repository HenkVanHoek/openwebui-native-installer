🚀 Open WebUI Native Installer

This is a minimal Ubuntu-based native installer for Open WebUI (with Ollama backend support).

⚙️ Prerequisites

Make sure your system meets the following requirements before installing:

  • ✅ Ubuntu Server 22.04 or newer
  • ✅ Outbound internet access (for downloading packages and dependencies)
  • ✅ curl installed
    sudo apt update && sudo apt install curl -y
  • ✅ VM must be able to connect to your Ollama host on port 11434
    • On your Windows PC, ensure Ollama is running and listening at:
      http://<host-ip>:11434
    • Your VM should use bridged or host-only networking to reach this IP and port.

🔧 Quick Install

Run this command on your Ubuntu VM:

curl -sL https://henkvanhoek.github.io/openwebui-native-installer/install.sh | bash
Download install.sh
